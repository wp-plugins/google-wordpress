== Description ==

Include your latest Google+ posts on your site with this handy widget. More features (including importing posts directly into WordPress) coming soon!

I'm open to any feedback and suggestions.

== Installation ==

1. Upload `/google-wordpress/` directory to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the Google+ WordPress settings page and enter your Google API key
1. Add the widget to your sidebar and be sure to set your user id (the number found in your profile URL) and the maximum number of posts to display

The venue's ID can be found as a number at the end of the venue's URL on Foursquare.
